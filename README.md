# Financial Fraud Detection

## Objective
Detect fraudulent credit card transactions using ensemble methods and anomaly detection.

## Dataset
- Total Transactions: 284,807
- Fraud Cases: 492 (0.17%)
- Normal Cases: 284,315

## Algorithms Used
- Random Forest Classifier (Ensemble Method)
- Isolation Forest (Anomaly Detection)

## Results
- Random Forest Accuracy: 99.96%
- Isolation Forest Accuracy: 99.76%
- Top Fraud Indicators: V17, V12, V14

## Tech Stack
Python, Pandas, Scikit-learn, Matplotlib
